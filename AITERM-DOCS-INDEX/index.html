    Quick Reference

    Key Commands

    # Installation
uv pip install git+https://github.com/Data-Wise/aiterm.git

# Verification
aiterm doctor

# Context detection
aiterm detect

# Profile management
aiterm profile list
aiterm profile switch PROFILE

# Claude Code integration
aiterm claude approvals list
aiterm claude approvals set PRESET
aiterm claude settings show
